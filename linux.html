🐧 Linux Command Center 2026

Guía interactiva para dominar la terminal desde cero.

Update & Cleanup

Mantenimiento total: actualiza la lista de software, instala mejoras y borra archivos residuales.

sudo apt update && sudo apt upgrade -y && sudo apt autoremove

ls -laS

Lista todos los archivos (incluso ocultos) ordenados por tamaño de mayor a menor.

ls -laS --block-size=M

pwd

Print Working Directory: Muestra la ruta completa de tu ubicación actual.

pwd

grep -r

Busca un texto específico dentro de todos los archivos de una carpeta de forma recursiva.

grep -r "texto_a_buscar" /ruta/carpeta

tar -xvf

Descomprime archivos con formato .tar de manera rápida.

tar -xvf archivo.tar

chmod +x

Da permisos de ejecución a un archivo (útil para scripts .sh).

chmod +x script.sh

ps aux --sort

Muestra los procesos que más memoria RAM están consumiendo ahora mismo.

ps aux --sort=-%mem | head -n 10

journalctl -xe

Muestra los últimos errores detallados del sistema. Ideal para debug.

journalctl -xe

mtr

Combina ping y traceroute para analizar la calidad de tu conexión a un host.

mtr google.com
