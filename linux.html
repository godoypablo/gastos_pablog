🐧 Master Linux: Guía de Comandos 2026

Tutorial dinámico para alumnos. Selecciona una categoría para filtrar.

ls

Lista el contenido de un directorio.

ls -la /var/www

pwd

Muestra la ruta del directorio actual.

pwd

cd

Cambia de directorio. 'cd ..' sube un nivel.

cd /home/usuario/descargas

locate

Busca archivos en la base de datos del sistema.

locate -i archivo.txt

find

Busca archivos en tiempo real en un directorio.

find . -name "config.php"

mkdir / rmdir

Crea o elimina directorios (si están vacíos).

mkdir nueva_carpeta

rm

Elimina archivos o directorios completos.

rm -rf carpeta_con_archivos

cp / mv

Copia o mueve (y renombra) archivos.

mv viejo.txt nuevo.txt

touch

Crea un archivo vacío rápidamente.

touch index.html

cat / grep

Muestra contenido o busca texto dentro de él.

cat log.txt | grep "error"

nano / vi

Editores de texto por consola.

nano config.txt

tar / zip

Comprime o empaqueta archivos.

tar -czvf backup.tar.gz /data

sudo

Ejecuta comandos como superusuario (root).

sudo apt update

chmod

Cambia permisos (Lectura, Escritura, Ejecución).

chmod 755 script.sh

chown

Cambia el dueño de un archivo o grupo.

chown usuario:grupo archivo

useradd / userdel

Crea o borra usuarios del sistema.

sudo useradd -m nuevo_profe

whoami

Muestra el usuario con el que estás logueado.

whoami

top / htop

Monitor de procesos y recursos (CPU/RAM).

htop

df / du

Muestra espacio en disco y tamaño de carpetas.

df -h

systemctl

Gestiona servicios (iniciar, detener, reiniciar).

sudo systemctl restart apache2

uname -a

Información del kernel y del sistema.

uname -a

kill

Termina un proceso mediante su ID (PID).

kill -9 1234

history

Lista los comandos ejecutados recientemente.

history | tail -n 20

ping

Verifica latencia y conexión con un host.

ping google.com

wget / curl

Descarga archivos o transfiere datos vía URL.

wget https://site.com/file.zip

ip address

Muestra interfaces de red y direcciones IP.

ip addr show

scp / rsync

Copia archivos entre servidores de forma segura.

rsync -avz local/ remoto:/backup

nslookup / dig

Consulta registros DNS de un dominio.

dig A midominio.com
