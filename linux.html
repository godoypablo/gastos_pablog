🐧 Guía Total de Linux para Alumnos

Desde navegación básica hasta rutinas de mantenimiento profesional.

Mantenimiento

Actualización Total

Actualiza la lista de repositorios y luego instala todas las mejoras disponibles sin preguntar.

sudo apt update && sudo apt upgrade -y
Limpieza

Limpieza de Espacio

Borra instaladores antiguos y dependencias que ya no se usan para liberar disco.

sudo apt autoremove && sudo apt autoclean
Salud

Estado del Sistema

Rutina para ver memoria libre, espacio en disco y temperatura (si está disponible).

free -m && df -h && uptime

ls

Lista archivos y carpetas. Usa '-la' para ver archivos ocultos y detalles.

ls -la

pwd

Muestra la ruta absoluta de la carpeta donde estás trabajando ahora.

pwd

cd

Cambia de directorio. 'cd ..' para retroceder, 'cd ~' para ir al inicio.

cd /ruta/del/directorio

locate

Busca la ubicación de un archivo rápidamente usando la base de datos.

locate mi_archivo.txt

find

Busca archivos en tiempo real filtrando por nombre, fecha o tamaño.

find . -name "*.log"

mkdir

Crea una nueva carpeta o directorio.

mkdir nombre_carpeta

rm

Borra archivos o carpetas. '-r' borra carpetas con todo su contenido.

rm -rf carpeta_basura

cp

Copia archivos o directorios de un lugar a otro.

cp origen.txt destino.txt

mv

Mueve un archivo o le cambia el nombre.

mv archivo.txt nueva_ruta/

nano / vi

Abre editores de texto directamente en la terminal para modificar archivos.

nano configuracion.conf

cat

Muestra todo el contenido de un archivo en la pantalla.

cat lista.txt

grep

Busca una palabra específica dentro de un archivo o salida de comando.

grep "ERROR" sistema.log

sudo

Ejecuta cualquier comando con permisos de administrador (Root).

sudo command_here

chmod

Cambia los permisos de un archivo (lectura, escritura, ejecución).

chmod 755 script.sh

chown

Cambia el propietario y el grupo de un archivo o carpeta.

sudo chown usuario:grupo archivo

passwd

Permite cambiar la contraseña del usuario actual o de otro.

sudo passwd nombre_usuario

top / htop

Muestra el uso de CPU, RAM y los procesos que están corriendo.

htop

df -h

Muestra el espacio total, usado y disponible en tus discos.

df -h

ps aux

Muestra una "foto" de todos los procesos activos en el sistema.

ps aux | grep mysql

systemctl

Controla servicios del sistema (iniciar, parar o ver estado).

sudo systemctl status ssh

shutdown

Apaga o reinicia el sistema de forma segura.

sudo shutdown -r now

ping

Envía paquetes a una IP para ver si responde y cuánta latencia hay.

ping 8.8.8.8

ip addr

Muestra tus tarjetas de red y las direcciones IP asignadas.

ip addr show

wget

Descarga archivos directamente desde internet a la carpeta actual.

wget https://wordpress.org/latest.zip

netstat -tuln

Muestra todos los puertos que están abiertos en tu máquina.

sudo netstat -tuln
