🐧 Linux Interactivo 2026

Copia comandos con un clic y pégalos en tu terminal (Shift + Insert).

Actualizar Sistema

Actualiza repositorios y software instalado.

sudo apt update && sudo apt upgrade -y

Limpiar Basura

Elimina archivos temporales y paquetes huérfanos.

sudo apt autoremove && sudo apt autoclean

Listar Detallado

Muestra archivos con permisos, dueño y tamaño legible.

ls -lah

Buscar Archivo

Busca un archivo por nombre en el directorio actual.

find . -name "archivo.txt"

Espacio en Disco

Muestra el uso de particiones en formato humano (GB/MB).

df -h

Memoria RAM

Muestra memoria libre y usada en Megabytes.

free -m

Mi IP Pública

Consulta rápidamente qué IP ve internet.

curl ifconfig.me
